# Python Writing Pad APP

A simple Writing Pad application built using Python and Tkinter.

Current Version: 'v1.0.0'

## Features
- Create new files
- Open existing files
- Save files
- Edit files
- Font Settings
- Simple GUI

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python
2. Run:
'''bash
python main.py
'''
